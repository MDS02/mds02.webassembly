# MDS02.WebAssembly

Op deze pagina zitten twee functies: "oppervlakte" en "rekenmachine (will be added soon)"

Om beide functies te kunnen bereiken is het een kwestie van de Javascript console openen, en daar de functie typen.

Voorbeelden:

oppervlakte(7)
-> 49

rekenmachine(10,20,4)
->0.5

Rekenmachine heeft 4 verschillende functies:

- Functie 1 = optellen
- Functie 2 = aftrekken
- Functie 3 = keer
- Functie 4 = gedeeld door
